CSS基础

Posted on 2021-06-03 In 前端 , CSS Views:
Symbols count in article: 8.3k

层叠样式表（Cascading Style Sheets，CSS）：定义网页的表现，美化网页。具体来说，就是用于设置网页中元素（标签）的样式。

优点：

  • 内容（HTML）与表现（CSS）分离
  • 可以实现复用
  • 样式丰富

定义 CSS 的三种方式

行内样式

在标签内部通过 style 属性设置标签的样式

问题：只能控制当前的 <p> 标签

内部样式

在 <head> 标签中通过 <style> 标签定义样式

问题：可以控制当前 html 页面中的所有 <p> 标签的样式，但是不能在其他 html 页面中进行复用。

外部样式

创建单独的 css 文件，并在其中定义样式，然后在 html 页面中引入

优点：编写外部样式表，不仅可以进行复用，而且可以使用浏览器的缓存机制，从而加快网页的加载速度，提升用户的体验。

优先级：**就近原则**，哪个样式离选择器最近，就使用哪个样式。

选择器

作用：选择页面中某一个或某一类元素。

基本选择器
标签选择器

用于选择一类标签，如：选择所有的 <p> 标签

类选择器

用于选择 class 属性值一致的标签，可以跨标签

PS：class 属性可以有多个值

ID 选择器

根据 id 属性的值选择标签，全局唯一

优先级：id > class > 标签

复合选择器
交集选择器

语法：选择器1选择器2选择器3 {}

并集选择器

语法：选择器1, 选择器2, 选择器3 {}

  • 后代元素
  • 兄弟元素

属性选择器

语法：

  • [属性]{}：选中包含该属性的所有标签
  • p[属性]{}：选中包含该属性的所有 p 标签
  • p[属性="属性值"]：选中“属性=‘属性值’”的 p 标签
  • p[属性^="属性值"]：选中以属性值开头的 p 标签
  • p[属性$="属性值"]：选中以属性值结尾的 p 标签
  • p[属性*="属性值"]：选中包含属性值的 p 标签
伪类选择器

伪类一般用 : 开头，如：

  • :first-child：第一个子元素

  • :last-child：最后一个子元素

  • :nth-child()：第 n 个子元素

    :nth-child(n) ：所有子元素

    :nth-child(2n)：偶数位子元素

    :nth-child(2n+1)：奇数位子元素

ps：以上这些伪类根据所有子元素排序

如：body p:first-child {}，如果 body 的第一个子元素不是 p，那这个 css 样式就不起作用

ps：以上这些伪类则是根据标签类型进行排序

  • :not(:first-child)

如：body p:first-of-type {}，就会对 body 下的第一个 p 标签起作用，不管 p 标签前面是否有其他标签。

盒模型

CSS 盒模型本质上是一个盒子，封装周围的HTML元素，它包括：边距，边框，填充，和实际内容。

  • Margin(外边距) - 清除边框外的区域，外边距是透明的。
  • Border(边框) - 围绕在内边距和内容外的边框。
  • Padding(内边距) - 清除内容周围的区域，内边距是透明的。
  • Content(内容) - 盒子的内容，显示文本和图像。

  • width 和 height 用于设置盒子 Content 的大小

  • border-width、border-color、border-style 用于设置盒子 Border 的样式

  • border-width: 10px 20px 30px 40px：

    • 四个值：上 右 下左
    • 三个值：上 左/右 下
    • 两个值：上/下 左/右
    • 一个值：上下左右

类似属性：border-top-width、border-bottom-width、border-left-width、border-right-width

要分别显示定义 border-width、border-color、border-style 会特别麻烦，可以简写：

border: 5px red solid ，width、color、style 没有先后顺序

元素水平方向上的布局：

一个元素在其父元素中水平方向上的位置由以下几个属性共同决定：

  • margin-left
  • border-left
  • padding-left
  • width
  • padding-right
  • border-right
  • margin-right

如果父元素中只有一个子元素，需要满足以下等式：

margin-left + border-left + padding-left + width + padding-right + border-right + margin-right = 父元素的 width

如果以上等式不满足，浏览器会自动跳转 margin-right 的值以使等式成立。
